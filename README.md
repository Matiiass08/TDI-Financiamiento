## Stack
- React + Vite + TypeScript
- **Zustand** (estado global)
- **Recharts** (gráficos torta/barras)
- **TailwindCSS** (UI)


> Elegimos **Zustand** por su simplicidad y bajo boilerplate para un MVP; y **Recharts** por su enfoque declarativo y responsivo.
